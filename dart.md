## 1. 空学习

| 命令        | 含义                                                                         |
| ----------- | ---------------------------------------------------------------------------- |
| **a ??= 3** | a 为 null 则赋值 a=3，a 不为 null 则该操作不执行                             |
| **??**      | 运算符左边的表达式返回的是空值，则会计算并返回右边的表达式。                 |
| **A?.B**    | 如果 A 等于 null，那么 A?.B 为 null,如果 A 不等于 null，那么 A?.B 等价于 A.B |
